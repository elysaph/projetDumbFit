# DumbFit - Suivi de Santé Intelligent

## Description

DumbFit est une application de suivi de santé qui permet aux utilisateurs de monitorer leurs objectifs de bien-être avec une logique de tranches de performance.

## Fonctionnalités Principales

- **Suivi des Objectifs** : Définissez vos objectifs de santé (sommeil, activité physique, nutrition, hygiène, etc.)
- **Système de Tranches** : Vos performances sont évaluées selon des tranches prédéfinies :
  - 🟢 **Optimal** : Vous êtes dans la bonne fourchette
  - 🟡 **À surveiller** : Vous êtes en dehors de la cible depuis peu
  - 🔴 **Alerte** : Vous déviez significativement de votre objectif depuis un moment et une suggestion s'affiche pour contacter un coach professionnel

- **Avertissements et Recommandations** :
  - Si votre performance sort de la tranche d'objectif, vous recevez un avertissement
  - Une suggestion automatique s'affiche pour contacter un coach professionnel
  - Accès direct aux profils de coaches disponibles
